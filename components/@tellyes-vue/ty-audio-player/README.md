### ty-audio-player
* 不支持H5
#### 使用说明
  | 参数 | 说明 | 默认值 |
  |---|---|---|
  | url | 资源链接 | `` |
  | vpId | 当多个组件共存时，需要用id区分组件。 | `` |
  
#### 示例
  ```html
  <tyAudioPlayer 
  url="http://114.116.110.224:3005/producerResource/fc/fc740262b0f79353463c9912e176e9de.mp3" 
  vpId="fc740262b0f79353463c9912e176e9de">
  </tyAudioPlayer>
  ```
